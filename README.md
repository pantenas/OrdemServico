# Ponto das Antenas — Gerenciamento de Ordens de Serviço

Sistema moderno e premium para gestão de ordens de serviço, focado em prestadores de serviços de instalação e manutenção.

## ✨ Funcionalidades

- **Dashboard Interativo**: Gráficos de desempenho (Chart.js) e KPIs em tempo real.
- **Controle de Acesso (RBAC)**: Perfis diferenciados para Administradores, Vendedores e Técnicos.
- **Gestão de Fotos**: Upload de até 5 fotos por OS com compressão automática.
- **Integração WhatsApp**: Botões de contato direto com o cliente via link `wa.me`.
- **Visão do Técnico**: Interface simplificada com gráfico de desempenho pessoal.
- **Impressão Térmica**: Geração de cupons otimizados para impressoras de 50mm.
- **Persistência Local**: Armazenamento seguro via `localStorage`.

## 🚀 Como Executar

O projeto é uma aplicação web estática (Vanilla JS). Não requer instalação de dependências complexas.

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador moderno.
3. Use as credenciais cadastradas (padrão Admin):
   - **E-mail**: `admin@pontodasantenas.com`
